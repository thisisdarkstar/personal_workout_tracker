# Personal Workout Tracker

A 12-week boxing + calisthenics workout tracker mobile app with diet tracking, water intake, daily notes, and progress analytics.

## Features

- **12-Week Program** - 3 phases: Foundation, Building, Peak
- **Daily Workouts** - Strength, conditioning, boxing drills, rest days
- **Diet Tracking** - 7 meals per day with checklist
- **Water Tracking** - Customizable daily target (glasses)
- **Daily Notes** - Journal your progress and feelings
- **Calendar View** - Navigate any day in the program
- **Progress Stats** - Weekly completion %, streaks

## Tech Stack

- React + Vite
- Tailwind CSS (dark/neon boxing theme)
- localStorage persistence
- Mobile-first responsive

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5200

## Build

```bash
npm run build
```

## Data

All data stored in browser localStorage. No account required.
